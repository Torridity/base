Libraries folder for KIT Data Manager
-------------------------------------

This folder serves as local repository for KIT Data Manager builds. It contains all libraries that are not available in public repositories. Most of them were developed at 
the Institute for Data Processing and Electronics of the Karlstuhe Institute of Technology, some (org.fzk.globus.* and com.google.gwt.*) are developed by external institutions.
All of these libraries are published under the Apache2 License that can be found at 'licenses/apache2.license'.
